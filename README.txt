Neste projeto pegamos uma tabela CSV com as produções cientificas do programa de posgraduação DMMDC
e se crea o arquivo de correlação de autores para criar um arquivo .net, formato do gephi, que consegue 
visualizar redes complexas.

para este script se precisa ter instalado python 3.6 
e o arquivo csv de entrada com a mesma estrutura que o que esta no repositorio até a 4ta coluna
